## 🚀 Getting Started

Clone this repository:

```bash
git clone https://github.com/haticegun/python-practice-projects.git
```

Navigate to the repository:

```bash
cd python-practice-projects
```

Open the project you want to run:

```bash
cd project-folder
```

Run the application:

```bash
python3 main.py
```
